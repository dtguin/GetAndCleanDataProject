# CodeBook for the GetAndCleanDataProject
After running run_analysis() the file CleanAndTidy.txt will be created in your working directory.

Enter the following R command to read in the CleanAndTidy data:
    clean_tidy_df <- read.table("CleanAndTidy.txt")
